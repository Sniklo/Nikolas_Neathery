Dynamic Workout Machine

The goal of this project was to create a workout machine that could be used for both anarobic and arobic exercises. Meaning that it could be used for both high weight low rep and low weight high rep. This was done by controlling the pressure in a piston as resistance for the machine. For this project I was responsible for design and manufacturing. Design was done using onshapes sheet metals tools and for manufacturing I utilized a laser cutter for the majority of the structure, a lathe for all the pulley axels and a 3D printer for the pulleys.

RBE: 3001

WPI structures the courses such that the beginning ones introduce mechanical, electrical and software skills while the later courses focus on less broad skills within these branches. RBE: 3002

Following this was the introduction to ROS. The goal of this project was to navigate a maze using A* and SLAM for localization. For this we used turtle bots with lidar. To start, everything was done in SIM then was migrated to a physical robot. This demonstrated that this transition is rarely seamless. The first challenge of the project was figuring out how to map the unknown area. We accomplished this by approaching the largest unknown area or frontier and reassessing the area until there were no longer unknown areas or frontiers. Once this was completed the robot was placed in a random location and needed to localize itself then drive to a desired position.

RBE: 450X

The latest project I completed was using a 2 DOF arm and a camera to find an ideal grasp for an object. The camera returned RGB D data in the form of a point cloud. This was again done using ROS 2 and was only in simulation. Once we had the point cloud, the top layer of the object was separated and from here an optimal grasp was derived. When I took it was in the experimental stage, and had never been taught with the use of ROS 2. For me this was my first time using ROS 2 and I not only became familiar with point clouds, edge detection and grasp derivation but also how to convert libraries meant for ROS 1 to ROS 2.
