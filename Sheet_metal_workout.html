Dynamic Workout Machine

overview

For this project myself and 3 other team members were tasked with creating a workout machine that utilizes a piston as resistance for a workout machine. We were given designs for a simple rowing machine but we wanted to go further and do somthing a bit more complex. In the end we designed a cable machine that can be used for multiple different exercises and an interface that showed the user metrics like power, reps and current estimated weight based on the pressure in the piston

How it works

The machine was designed using Onshape's sheet metal tools, enabling seamless work on multiple parts simultaneously. The first challenge was extending the piston's stroke length to achieve the necessary range of motion, which was resolved using a pulley reduction system. The piston was attached to a small pulley, and the machine's output was connected to a large pulley, with cables routed through a series of idler pulleys. Manufacturing custom all-metal pulleys was impractical and buying off-the-shelf parts was not an option due to cost constraints. The solution was homemade pulleys combining laser-cut steel plates and 3D printed drums. Entirely 3D printed pulleys risked insufficient strength, and solely laser-cut pulleys were too complex. The hybrid design utilized 3D printing for the pulley profile and groove, while laser-cut plates provided reinforcement. Bolts connected the 3D printed part to the steel plates, ensuring durability and effective force transfer.
