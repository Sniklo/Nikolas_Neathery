Dynamic Workout Machine

Overview

This project focused on designing and manufacturing a multi-purpose cable-based workout machine that uses a pneumatic piston as its resistance element. Starting from an initial concept of a simple rowing mechanism, our team extended the design into a configurable cable machine capable of multiple exercises while maintaining a robust and manufacturable structure.

In addition to the mechanical system, we integrated a measurement and user interface stack that provides real-time workout metrics such as repetitions, session duration, estimated lifted weight (derived from piston pressure), and mechanical power output.

How it Works

Stroke Extension via Pulley Ratio

The piston had a limited stroke length, so we implemented a reduction system using two drums of different diameters mounted on a shared axle. The piston drives the smaller drum, while the output cable wraps around the larger drum. This creates approximately a 1:1.5 motion ratio, effectively increasing bar travel relative to piston stroke.

Hybrid Pulley Manufacturing

Custom all-metal pulleys were not feasible under project constraints. The solution was a hybrid design using laser-cut steel plates for structural integrity and 3D-printed drums for the cable groove. Bolts clamp the assembly together, ensuring strength and reliable force transfer.

Measurement & UI

An Arduino Uno collects signals from a potentiometer (for pulley rotation) and a pressure sensor (for piston force). Data is streamed over serial to a Python (PyQt) interface that calculates reps, estimated weight, session duration, and mechanical power in real time.

Testing & Results

A test session of five full-range pulls at 3 bar confirmed expected relationships between pressure, equivalent lifted weight, and power output. The pulley ratio successfully extended the range-of-motion beyond the piston stroke.
