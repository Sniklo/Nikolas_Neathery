Mobile Soft Robot – Pneumatic Crawling Robot

Mold Design & Manufacturing

The robot was manufactured using a multi-stage mold process to replace the original multi-material 3D printing approach. Each triangular prism was cast individually, then reinforced with a stiffer silicone layer before modular assembly.

Modular row assembly

Project Overview

This project focused on replicating a crawling soft robot originally developed at Harvard University, while replacing the original multi-material 3D printing process with a modular mold-based approach.

Locomotion Mechanism

  1. On Inners – Off Externals: Inner rows contract and lift.
  2. On Inners – On Externals: Outer rows contract and lower structure.
  3. Off Inners – On Externals: Inner rows reinflate and push forward.
  4. Off Inners – Off Externals: Full reset and stabilization.

Performance Results

  • 10.6 mm/s on rigid wood surface
  • 7.24 mm/s on foam surface
  • 5× velocity increase after pneumatic redesign

Conclusion

The project demonstrated that advanced soft robotic locomotion can be achieved without specialized multi-material 3D printers. Through modular mold fabrication and pneumatic optimization, a functional crawling soft robot was successfully built.
