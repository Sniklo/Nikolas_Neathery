Mobile Soft Robot – Pneumatic Crawling Robot

Course Project | Soft Robotics | University of Southern Denmark

This project focused on replicating a pneumatically actuated soft crawling robot originally developed at Harvard University, while replacing the original multi-material, multi-nozzle 3D printing process with a modular mold-based manufacturing approach.

Mold Design & Manufacturing

The robot was fabricated using a multi-stage silicone molding process. Hollow triangular prisms were cast using 3D-printed molds to ensure airtight internal cavities. A second mold stage was used to add a stiffer silicone layer to one face of each prism, creating asymmetric bending behavior.

The structure was built modularly, with independent rows fabricated separately to reduce risk of total failure due to manufacturing defects. This approach replaced the original fully integrated multi-nozzle print.

Locomotion Mechanism

The robot consists of rows of hollow triangular silicone prisms arranged into two groups:

  • Inner rows (central modules)
  • Outer rows (external modules)

Each prism bends directionally due to asymmetric stiffness. When vacuum is applied (On), the internal cavity collapses and the prism bends toward the softer side. When vacuum is released (Off), the structure reinflates and returns to its original shape.

Four-Stage Crawling Cycle

  1. Stage A — On Inners / Off Externals: The inner rows contract, lifting and shifting the central body forward.
  2. Stage B — On Inners / On Externals: The outer rows contract, anchoring the robot and stabilizing its structure.
  3. Stage C — Off Inners / On Externals: The inner rows reinflate while outer rows remain contracted, generating forward propulsion.
  4. Stage D — Off Inners / Off Externals: Both groups reinflate and reset the structure for the next cycle.

Control & Pneumatic System

The system uses an external vacuum generator for contraction and Arduino-controlled pumps for inflation. Valve states are managed through a timed state-machine sequence.

  • Inflation/deflation timing
  • Vacuum strength
  • Silicone stiffness
  • Surface friction

Performance Results

  • 10.6 mm/s on wood
  • 7.24 mm/s on foam
  • 5× velocity increase after pneumatic optimization

Conclusion

This project demonstrated that advanced soft robotic locomotion can be achieved without specialized multi-material 3D printing. Through modular mold fabrication and pneumatic optimization, a functional crawling soft robot was successfully created.
