Two axis vision based turret

overview

One of my ongoing projects is a 3D printed turret that is controlled via an arduino and openMV Camera. The goal of the project is to be able to track a target and fire some form of projectile at the target. The target being anthing from an april tag to my roommate.

How it works

As mentioned above the turret is controlled by an arduino that interfaces with an openMV cam. The camera is responsible for find the target, draw a box around it and report the location of the box to the ardiuno via I2C. The arduino is responsible for controlling the two motors that change the X and Y axis. To prevent over rotation of the X axis an encoder is used to monitor the current rotation. The Y axis utilizes current monitoring to detect when a limit is reached.
