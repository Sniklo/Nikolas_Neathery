RBE: 450X

overview

The latest project I completed was using a 2 DOF arm and a camera to find an ideal grasp for an object. The camera returned RGB D data in the form of a point cloud. This was again done using ROS 2 and was only in simulation. Once we had the point cloud, the top layer of the object was separated and from here an optimal grasp was derived. When I took it was in the experimental stage, and had never been taught with the use of ROS 2. For me this was my first time using ROS 2 and I not only became familiar with point clouds, edge detection and grasp derivation but also how to convert libraries meant for ROS 1 to ROS 2.

How it works

My solution to finding an adiquate grasp matrix was to first turn the 3D point cloud into a 2D image. From here a line was drawn between every point on the edge of the object. The goal for this project was to generate a grasp for a two fingered gripper, so the optimal grasp would grab the object on either side of the center of mass. Once every possible line was plotted I calculated the distance between the center of each line and the center of mass of the object. This meant that the line with the smallest distance from the center of mass was the most optimal grasp with the contact points being at each end point of the line
